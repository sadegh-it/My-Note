# My-Note
My Note یک اپلیکیشن نوت‌برداری مدرن است که با معماری MVVM، تزریق وابستگی Dagger Hilt، پایگاه داده Room، ذخیره‌سازی تنظیمات با DataStore و مسیر‌یابی با Navigation Compose ساخته شده است. این پروژه از Jetpack Compose برای طراحی رابط کاربری و از ساختار Clean و ماژولار برای توسعه‌پذیری بهتر استفاده می‌کند.
