# My Note

یک اپلیکیشن نوت‌برداری مدرن با معماری استاندارد و بهترین شیوه‌های توسعه اندروید.

## ✨ ویژگی‌ها
- طراحی با **Jetpack Compose**
- معماری **MVVM**
- تزریق وابستگی **Dagger Hilt**
- پایگاه داده **Room** برای ذخیره‌سازی نوت‌ها
- **DataStore** برای ذخیره تنظیمات (مثل تم دارک و لایت)
- **Navigation Compose** برای مدیریت مسیرها
- ذخیره‌سازی آفلاین و سریع
- رابط کاربری مدرن، ساده و قابل توسعه

## 🧱 تکنولوژی‌ها و کتابخانه‌ها
- Kotlin
- Jetpack Compose
- Room Database
- DataStore Preferences
- Dagger Hilt
- Navigation Compose
- Coroutines & Flow
- AndroidX Lifecycle

## 🎯 هدف پروژه
هدف این پروژه ساخت یک اپ نوت‌برداری کامل و قابل توسعه است که:
- از معماری استاندارد استفاده کند  
- به‌عنوان یک پروژه آموزشی و قابل انتشار کاربرد داشته باشد  
- قابلیت توسعه برای امکانات بیشتر را داشته باشد
